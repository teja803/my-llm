# 🤖 LLM Fine-Tuning Studio

LLM Fine-Tuning Studio is an interactive web application built using Streamlit that enables users to fine-tune Large Language Models (LLMs) for text classification and text generation tasks with real-time monitoring and evaluation.

## 🚀 Features
- Fine-tune popular transformer models:
  - BERT
  - DistilBERT
  - RoBERTa
  - GPT-2
  - DialoGPT
- Real-time training visualization with Plotly
- Interactive model testing interface
- Batch evaluation support
- Model save and download functionality
- GPU/CPU system monitoring
- User-friendly Streamlit dashboard

## 🛠️ Technologies Used

- Python
- Streamlit
- PyTorch
- Hugging Face Transformers
- Plotly
- Pandas

## 📊 Functional Modules

1. Dataset Upload & Validation
2. Model Configuration
3. Real-time Training Dashboard
4. Model Testing & Evaluation
5. Model Management & Export

## 🎯 Objective

The objective of this project is to simplify the process of LLM fine-tuning by providing an intuitive interface for training, monitoring, evaluating, and managing transformer-based NLP models without requiring complex coding workflows.
